1. Rechercher les médecins/pharmacies par nom ou ID, ou directement par ID de la visite.

2. Cliquer sur "Search". Une fois les résultats affichés, sélectionner les visites que vous souhaitez transférer.

3. Cliquer sur le bouton "Transfert".

4. Rechercher par nom ou ID et choisir le médecin / pharmacie vers lequel vous souhaitez transférer les visites sélectionnées. Ensuite, cliquer sur "Changer". Les visites seront transférées.
